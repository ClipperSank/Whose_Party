> ##### <font>---------- 腳踏車怎麼生? -----------</font>
>> * 找要畢業、認識的學長姊們買二手車。
>> * 上出清台大或是公館附近的二手貨社團找二手車。
>> * 去公館或後門附近的車行買全新的， google找的到 。
>> * 去排水源阿伯由報廢車輛所拼裝的拼裝二手車。
>> * 自己想辦法從家裡載來、或是從家裡騎上來。
>> * P.S. 要去水源辦 車證 被拖才不會找不到 。
>> * P.S. 不要亂停車， 車被拖吊 真的會很煩躁。
